Network configuration, cont.

  • use command - static IP

$ ifconfig
$ route
# ifconfig wlan0 inet xxx.xxx.xxx.xxx netmask xxx.xxx.xxx.xxx
$ route
# route add -net default gw xxx.xxx.xxx.xxx
$ ifconfig
$ route

Network configuration, cont.

  • use command - dynamic IP

# dhclient -v wlan0

Network configuration, cont.

  • modify /etc/network/interfaces - static IP

auto eth0
iface eth0 inet static
address xxx.xxx.xxx.xxx
netmask xxx.xxx.xxx.xxx
gateway xxx.xxx.xxx.xxx

  • # /etc/init.d/networking restart

Network configuration, cont.

  • modify /etc/network/interfaces - dynamic IP

auto eth0
iface eth0 inet dhcp

  • # /etc/init.d/networking restart

Network File System

RPC - Remote Procedure Call

  • portmapper, port 111

  • NFS server 啟動時會向 RPC 註冊其 File system 連線用的port
  • 提供NFS其中 File system 的 port 給Client
  • 若要使用NFS，server& client都要啟動RPC service
  • nfs client - nfs-common
  • nfs server - nfs-kernel-server

什麼是檔案系統

  • 檔案系統 是指儲存在電腦中的檔案與目錄

  • 不同的作業系統，所支援的檔案系統不一定相同，有部份的

  • 作業系統會因為特殊的理由，而採購獨有的檔案系統

  • 這些不同格式的檔案系統，則稱為 檔案系統類型

  • 以 Linux 作業系統的核心而言，目前已支援相當多種不同類型的檔案系統， 這些不同的檔案系統類型，主要是用來決定資訊該如何儲存到儲存媒體上

Linux 檔案系統與目錄管理

  • / 根目錄
  • /bin 二進位檔 (binary) 的縮寫，存放各種系統的執行檔
  • /boot 作業系統啟動時所需要的檔案，包含系統 kernel (vmlinuz)、 grub 開機程式等
  • /dev device 的縮寫，硬體裝置檔案，如硬碟、光碟、印表機
  • /etc 存放 Linux 系統與各服務的設定檔，如 apt, ssh, apache2, sudoers
  • /home 各使用者的家目錄

Linux 檔案系統與目錄管理

  • /lib 各種函式庫
  • /list+found 儲存檢測與修復檔案系統的資訊
  • /mnt 檔案系統的臨時掛載點
  • /media 卸除式媒體的掛載點
  • /opt 第三方軟體放置的目錄，不過有些第三方軟體也會放在 /usr/local 下
  • /proc 為 虛擬檔案系統 ，其中的資料都是存放在記憶體中，本身不佔硬碟空間。 裡面包含了系統 kernel 、process 等資訊

Linux 檔案系統與目錄管理

  • /root 為系統管理者 root 的家目錄
  • /sbin 存放只有 root 才能用來設定系統的執行檔
  • /sys 和 /proc 很像，也是一個 虛擬檔案系統 ，用來存放 kernel 模組， 及一些硬體資訊
  • /tmp 一般使用者皆可存取，也可供 process 放置暫存檔
  • /usr Unix Software Resource 的縮寫，存放各種軟體及其文件等資源
  • /var 主要存放各種經常會變動的檔案，如 cache, log, mail 等檔案

find

  • find 用來做檔案的搜尋
  • 用法
    • $ find 路徑 參數
  • -name 用來指定檔名
  • -type 用來指定檔案類型
  • -user, -group 用來指定搜尋檔案的擁有者或所在群組
  • -newer file 找到 mtime 比 file 還新的檔案
  • -exec 指令 {} \;

find

  • 找到所有 txt 檔案，並複製到 ~/txt 資料夾

    $ find . -name "*.txt" -exec cp {} ~/txt \;

  • {} 代表找到的檔案

  • \; 代表 -exec 的指令結束

  • x 解開 tar 檔
  • f 指定檔名
  • v 輸出過程
  • z 使用 gzip 壓縮，通常會把壓縮檔命名為 .tar.gz
  • j 使用 bzip 壓縮，通常會把壓縮檔命名為 .tar.bz2

正規表示法 Regular Expression

Outline

什麼是 Regular Expression

  • Regular expression (regexp)
  • 特殊的語法用來做
    • 字串比對
    • 字串取代
    • 取得特定的資訊

Regular Expression 的特殊字元

  • Basic regular expression (BRE)

    ^   $   .   *   \(   \)   [   \{   \}   \

  • Extended regular expression (ERE)

    |   ^   $   .   *   +   ?   (   )   [   {   }   \

Regular Expression 的特殊字元

  • ^ 行首
  • $ 行尾
  • . 比對一個任意字元

Regular Expression 的特殊字元

  • []

    • 比對 [] 裡的任一個字元

      [aeiou]         # 比對母音
[a-zA-Z0-9_]

Regular Expression 的特殊字元

  • [^]

    • 比對 [^] 之外的任一個字元

      [^aeiou]        # 比對不是母音的字

Regular Expression 的特殊字元

  • \ (backslash)

    • 跳脫字元，用來比對特殊符號

      \\                  # 比對 \
\[                  # 比對 [

Regular Expression 的特殊字元

  • |

    • 在 BRE 裡沒有 |

      a.c|a.b         # 比對 'abc', 'acb', ...

Regular Expression 的特殊字元

  • ()

    • 用來標記子字串，可用 \1 等取回 () 內的子字串

    • 在 BRE 裡是用 \(\) 表示

      (cat|dog)*

Regular Expression 的特殊字元

  • Perl 風格的 Regular Expression

  • \d 表示 [0-9]
  • \w 表示 [a-zA-Z0-9_]
  • \s 表示 [\r\n\t ]
  • \l 表示 [a-z]
  • \u 表示 [A-Z]

Regular Expression 重覆比對

  • {m,n}

    • 比對前一個東西重覆 m 次以上，不超過 n 次
    • 在 BRE 裡是用 \{\} 表示

  • *

    • {0,∞}

Regular Expression 重覆比對

  • +

    • {1,∞}
    • 在 BRE 裡沒有

  • ?

    • {0,1}
    • 在 BRE 裡沒有

在 grep 中使用 Regular Expression

  • 在 grep 搜尋樣式中可以使用 Regular Expression
    • 預設是 BRE
    • -E 使用 ERE
    • -P 使用 perl 樣式的 Regular Expression

在 grep 中使用 Regular Expression

$ grep "[0-9]\{3\}" *.txt

$ grep -E "[0-9]{3}" *.txt

$ grep -P "\d{3}" *.txt

sed

  • sed 是一個常用的文字檔分析處理工具
  • 以行作為處理的單位 (awk 可以把每行細分為數個欄位)

  • 使用 BRE

    $ sed "規則" filename

sed

  • 下面所說的刪除並不會真的更改檔案內容，只是在處理過程中刪掉，不顯示在螢幕上

  • 刪除 1 ~ 4 行，剩下的顯示出來

    $ sed "1,4d" file

  • 刪除含有三個數字的行

    $ sed "/[0-9]\{3\}/d" file

sed

  • 顯示含有三個數字的行，-n 取消預設印出所有行

    $ sed -n "/[0-9]\{3\}/p" file

  • 取代任意三個數字成 xxx

    $ sed "s/[0-9]\{3\}/xxx/" file

sed

  • 把任意三個數字用 [] 括起來，利用 \1 來取得 \(\) 裡的字串

    $ sed "s/\([0-9]\{3\}\)/[\1]/" file

  • 利用 g 來配對所有出現的樣式

    $ sed "s/\([0-9]\{3\}\)/[\1]/g" file

Mode

Mode

  • Normal Mode

    • 移動
    • 複製/貼上
    • 刪除

  • Insert Mode

    • 編輯文字

Mode

    • 選取文字

    • 執行 vim 指令
